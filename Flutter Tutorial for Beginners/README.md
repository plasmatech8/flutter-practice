## Widgets

There are many widgets and each widget can be configured using properties.

e.g:
* Text Widget
  * style
  * textAlign
  * overflow
  * maxLines
  * ...etc
* Button Widget
  * color
  * elevation
  * disabledColor
  * enabled
  * ...etc
* Row Widget
* Column Widget
* Image Widget
